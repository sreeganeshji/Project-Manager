Overview and usage:

The basic elements of this tool are tasks, task-groups and projects in the increasing order of hierarchy.

Create a project

Select Project from the homepage to create a new project. Provide the name and description. Proceed to adding your teammates. You can search based on their username or email from all the users. Once finished, press 'Done'.

The tasks that are part of the project can be grouped into task groups. You can create task groups in step2.

Create a task group

From the project detail page, select Task Group to create a new task group. Provide the details and proceed to adding your teammates from the project. A task group can contain many tasks.

Create a task

Select Task to create a new task. There are multiple stages for a task including "opened","in progress", "in review", "closed". Tasks can also be prioritized. All the tasks in a task group will be presented in the 'kanban board' format in the task group detail page.
